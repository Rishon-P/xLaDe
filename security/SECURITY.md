# Overview

xLaDe is experimental. Please follow these guidelines to report security issues.

- Do not open public issues for security vulnerabilities.
- Report security issues privately to the repository owner via GitHub's private security advisory flow or by emailing the maintainers (see CONTRIBUTORS.md for contacts).
- Include clear reproduction steps, affected versions (see VERSION), and any patches if available.

We will acknowledge receipt within 3 business days and aim to respond with a plan within 14 days.
